# 📈 Trading Analysis Telegram Bot

Бот для аналізу ринку акцій та валютних пар (включаючи OTC) з інтерфейсом у стилі торгових платформ.

## ✨ Функціонал
- Вибір між Акціями (Apple, McDonald's тощо) та Валютами (AUD/NZD).
- Вибір часу угоди (1-10 хвилин).
- Відображення графіків (імітація реального часу).
- Модуль технічного аналізу, що видає прогноз (Вгору/Вниз) та пояснює причину.

## 🚀 Встановлення
1. Клонуйте репозиторій.
2. Встановіть бібліотеки: `pip install -r requirements.txt`.
3. Створіть файл `.env` та додайте туди свій токен: `BOT_TOKEN=твій_токен`.
4. Запустіть бота: `python bot.py`.
